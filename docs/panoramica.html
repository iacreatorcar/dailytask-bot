Technical documentation — full application reference

Node.js + Express Claude AI (Haiku) Telegram Bot

📌 What is it

dailytask-bot is a self-hosted personal productivity dashboard with AI integration. It combines an advanced task manager with a Claude AI assistant, a Telegram bot for automatic reminders, Gmail inbox reading, backup monitoring, and job application tracking — all in a single dark-themed interface.

Backend: Node.js + Express  ·  Frontend: Vanilla HTML/CSS/JS  ·  Storage: local JSON files  ·  Port: localhost:3000

✨ Core features

📋 Task Manager

  • Create tasks with title, category, priority, and day
  • Priority levels: High / Medium / Low
  • Days: Today · Tomorrow · Someday
  • Subtasks to break down complex work
  • Quick notes per task
  • Per-task reminder time (HH:MM)
  • Move tasks between days easily

🤖 AI Assistant (Claude)

  • Generate structured tasks from free text
  • Suggest optimal priorities for the day
  • Plan up to 7 tasks for tomorrow
  • Break complex tasks into subtasks
  • Free chat with pending task context
  • Concise, practical responses

💼 Job Application Tracker

  • Kanban: Saved → Applied → Interview → Offer → Rejected
  • Add application with company, position, URL
  • Update application status
  • Telegram reminder for follow-ups
  • Monday alert for stale applications (> 3 days)

📬 Gmail Integration

  • Read unread emails via IMAP
  • 5-minute cache to avoid overload
  • Shows sender, subject, and date
  • Unread count in real time
  • Manual refresh available

💬 Telegram Bot

  • /oggi — today's tasks
  • /domani — tomorrow's tasks
  • /digest — morning summary
  • /jobs — active applications
  • /gmail — unread emails
  • /stats — statistics
  • /backup — backup status

⏰ Automatic Reminders

  • 08:00 — morning digest (today's tasks)
  • 21:00 — tomorrow's plan preview
  • Every minute — tasks with reminder time set
  • Sunday 10:00 — overdue backup alert
  • Monday 09:00 — stale applications alert

📊 Statistics

  • Total tasks: pending, completed, overall
  • Today's tasks with completion percentage
  • Tomorrow's pending count
  • Completed this week
  • Per-category progress bars

📄 CV Catalog

  • Auto-scan a configurable CV folder
  • Group files by company (from filename)
  • Icons per file type: .pdf, .doc, .docx
  • Smart filter for redundant keywords

💾 Backup Monitor

  • Track important folders
  • Status: ✅ Today · ⚠️ X days ago · ❌ Never
  • Mark last backup date
  • Telegram alert if overdue (> 7 days)
  • Add / remove monitored folders

🔄 How it works

1
Start

npm start — Express server starts on port 3000. Telegram bot begins polling.

2
Frontend

Browser opens localhost:3000 and loads the full interface from public/index.html.

3
Task management

User creates a task → frontend calls POST /api/tasks → backend saves to tasks.json.

4
AI Assistant

User clicks an AI button → frontend sends the prompt → backend calls Anthropic API → response appears in the chat panel.

5
Telegram Bot

User sends a command → polling bot receives it → executes logic → replies with Markdown-formatted messages.

6
Automatic cron jobs

Every minute/hour/day the scheduled jobs send morning digests, reminders, and weekly alerts via Telegram — no manual action needed.

🔌 API endpoints

Tasks

GET /api/tasks List all tasks
POST /api/tasks Create a new task
PUT /api/tasks/:id Update an existing task
DEL /api/tasks/:id Delete a task

Job Applications

GET /api/jobs List all applications
POST /api/jobs Add an application
PUT /api/jobs/:id Update application status
DEL /api/jobs/:id Remove an application

Gmail & AI

GET /api/gmail Read unread emails (5 min cache)
POST /api/gmail/refresh Force email refresh
POST /api/ai Call Claude AI with a prompt

Backup & Utilities

GET /api/backup List monitored folders
POST /api/backup/mark Mark last backup date
POST /api/backup/folders Add folder to monitor
DEL /api/backup/:folder Remove folder from monitoring
GET /api/cv-scan Scan CV folder
GET /api/stats Task and job statistics
