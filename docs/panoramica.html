📌 Cos'è l'applicazione

AI Daily Planner è un'applicazione web personale di gestione produttività con integrazione IA. Combina un task manager avanzato con un assistente Claude AI, un bot Telegram per reminder automatici, lettura email Gmail, monitoraggio backup e gestione candidature LinkedIn — tutto in un'unica interfaccia dark-themed.

Backend: Node.js + Express  ·  Frontend: HTML/CSS/JS vanilla  ·  Persistenza: file JSON locali  ·  Porta: localhost:3000

✨ Funzionalità principali

📋 Gestione Task

  • Crea task con titolo, categoria, priorità e data
  • Priorità: Alta / Media / Bassa
  • Date: Oggi · Domani · Prima o poi
  • Sotto-task (subtask) per scomporre il lavoro
  • Note rapide per ogni task
  • Promemoria con orario (HH:MM)
  • Sposta task tra giorni facilmente

🤖 Assistente IA (Claude)

  • Genera task strutturati da descrizione libera
  • Suggerisce le priorità ottimali del giorno
  • Pianifica un massimo di 7 task per domani

🔄 Come funziona (flusso)

1
Avvio

npm start — Server Express parte su porta 3000. Il bot Telegram inizia il polling.

2
Frontend

Il browser apre localhost:3000 e carica l'interfaccia completa da public/index.html.

3
Gestione Task

L'utente crea un task → il frontend chiama POST /api/tasks → il backend salva su tasks.json.

4
Assistente IA

L'utente clicca un pulsante IA → il frontend invia il prompt → il backend chiama l'API Anthropic → la risposta torna nella chat.

5
Telegram Bot

L'utente invia un comando → il bot in polling lo riceve → esegue la logica → risponde con messaggi Markdown formattati.

6
Cron Jobs automatici

Ogni minuto/ora/giorno i job schedulati inviano digest mattutini, reminder e avvisi settimanali via Telegram, senza intervento manuale.

🔌 API Endpoints

Task

GET /api/tasks Leggi tutti i task
POST /api/tasks Crea un nuovo task
PUT /api/tasks/:id Aggiorna un task esistente
DEL /api/tasks/:id Elimina un task

Candidature LinkedIn

GET /api/jobs Lista candidature
POST /api/jobs Aggiungi candidatura
PUT /api/jobs/:id Aggiorna stato candidatura
DEL /api/jobs/:id Rimuovi candidatura

Gmail & IA

GET /api/gmail Leggi email non lette (cache 5 min)
POST /api/gmail/refresh Forza refresh email
POST /api/ai Chiama Claude AI con prompt

Backup & Utility

GET /api/backup Lista cartelle monitorate
POST /api/backup/mark Segna data ultimo backup
POST /api/backup/folders Aggiungi cartella da monitorare
DEL /api/backup/:folder Rimuovi cartella dal monitoraggio
GET /api/cv-scan Scansione cartella CV
GET /api/stats Statistiche task e candidature
