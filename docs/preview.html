Features
Built for solo developers

All the tools a freelance dev or job-hunter needs, without the SaaS subscription.

✅

Smart Task Manager

Today / Tomorrow / Someday. Categories, priorities, subtasks, per-task reminder times. Claude AI can generate your whole task list from a single sentence.

💼

Job Application Kanban

Track every application through the pipeline: Saved → Applied → Interview → Offer. Auto-alerts on Monday for applications pending 3+ days.

🤖

Claude AI Assistant

Powered by claude-haiku-4-5. Generate tasks, prioritize your day, draft messages, break down complex work — all from a chat panel.

📲

Telegram Bot

Morning digest at 8am. Evening preview at 9pm. On-demand commands: /oggi /jobs /gmail /stats. No app needed — just Telegram.

📬

Gmail Integration

Read unread emails directly in the dashboard via IMAP. 5-minute cache. Ask the Telegram bot /gmail for a quick inbox summary anywhere.

💾

Backup Monitor

Track which project folders were last backed up and when. Weekly Sunday alert if anything is overdue. Simple, no cloud sync needed.

Get Started
Running in 4 steps

No Docker, no cloud setup. Clone, configure, run.

1
Clone & install git clone https://github.com/iacreatorcar/dailytask-bot.git cd dailytask-bot npm install
2
Configure your .env cp .env.example .env # Add your Anthropic API key, Telegram token, Gmail app password
3
Init data files cp data/tasks.example.json data/tasks.json cp data/jobs.example.json data/jobs.json cp data/config.example.json data/config.json
4
Run node server.js

Open http://localhost:3000 — send /start to your Telegram bot.
